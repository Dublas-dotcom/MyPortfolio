JavaScript Basics

JavaScript is a powerful programming language that can add interactivity to a website. It was invented by Brendan Eich in 1995 and has become one of the core technologies of the web.

What is JavaScript?

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and much more. While HTML and CSS are languages that give structure and style to web pages, JavaScript gives web pages interactive elements that engage a user.

Note: JavaScript is not the same as Java. They are two different programming languages with different purposes and capabilities.

Variables and Data Types

Variables are used to store information that can be referenced and manipulated in a program. JavaScript has three ways to declare variables: var, let, and const.

// Variable declarations var name = "John"; // Function-scoped variable let age = 30; // Block-scoped variable const PI = 3.14; // Constant value (cannot be reassigned) // Data types let message = "Hello"; // String let score = 100; // Number let isActive = true; // Boolean let person = null; // Null let data; // Undefined let colors = ["red", "green"]; // Array let user = { // Object name: "Alice", age: 25 };

Operators

JavaScript includes operators that perform operations on variables and values.

// Arithmetic operators let x = 10 + 5; // Addition let y = 10 - 5; // Subtraction let z = 10 * 5; // Multiplication let w = 10 / 5; // Division let remainder = 10 % 3; // Modulus (remainder) // Comparison operators let isEqual = (5 == "5"); // true (loose equality) let isStrictEqual = (5 === "5"); // false (strict equality) let isGreater = (10 > 5); // true let isLess = (10 < 5); // false // Logical operators let andResult = (true && false); // false let orResult = (true || false); // true let notResult = !true; // false

Control Flow

Control flow statements allow you to control the execution of your code based on conditions.

// If statement let temperature = 25; if (temperature > 30) { console.log("It's hot outside"); } else if (temperature > 20) { console.log("It's warm outside"); } else { console.log("It's cold outside"); } // Switch statement let day = "Monday"; switch(day) { case "Monday": console.log("Start of the work week"); break; case "Friday": console.log("End of the work week"); break; default: console.log("It's a regular day"); } // For loop for (let i = 0; i < 5; i++) { console.log("Iteration number: " + i); } // While loop let count = 0; while (count < 3) { console.log("Count: " + count); count++; }

DOM Manipulation

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page structure so that programs can change the document structure, style, and content.

Selecting Elements

JavaScript provides several ways to select elements from the DOM.

// Select elements by ID let header = document.getElementById("header"); // Select elements by class name let items = document.getElementsByClassName("item"); // Select elements by tag name let paragraphs = document.getElementsByTagName("p"); // Modern selector methods (CSS selectors) let element = document.querySelector(".className"); // Returns first match let elements = document.querySelectorAll("div.item"); // Returns all matches

Modifying Elements

Once you've selected elements, you can modify their content, attributes, and styles.

// Changing content element.innerHTML = "New HTML content"; element.textContent = "New text content"; // Changing attributes element.setAttribute("src", "image.jpg"); element.id = "newId"; // Changing styles element.style.color = "blue"; element.style.fontSize = "20px"; element.style.display = "none"; // Hide element element.style.display = "block"; // Show element // Adding and removing classes element.classList.add("new-class"); element.classList.remove("old-class"); element.classList.toggle("active"); // Toggle class

Event Handling

JavaScript can respond to user interactions through event handlers.
